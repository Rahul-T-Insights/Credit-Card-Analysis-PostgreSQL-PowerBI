# Credit-Card-Analysis-POSTGRE-SQL-POWER-BI


## Project Overview

This project analyzes credit card transaction and customer data to uncover business insights related to revenue trends, customer behavior, spending patterns, and delinquency risk.

Using SQL for data analysis and Power BI for dashboard creation, this project simulates a real-world banking analytics use case.

---

## Tools Used

- POSTGRE SQL
- Power BI
- CSV / Excel Data
- Data Cleaning
- Dashboard Design

---

## Dashboard Pages

### Transaction Dashboard

Analyzed:

- Total Revenue
- Quarterly Revenue Trend
- Revenue by Card Category
- Top States by Revenue
- Revenue by Expense Type

### Customer Dashboard

Analyzed:

- Total Customers
- Average Income
- Average Credit Limit
- Customer Age Distribution
- Revenue by Occupation
- Delinquency by Card Type

---

## Key Insights

- Blue card category generated highest revenue.
- Q1 was the highest-performing quarter.
- Bills and Fuel were top spending categories.
- Businessman segment contributed highest revenue.
- Blue and Silver cards had higher delinquency rates.
